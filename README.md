#  경기자동차과학고 오늘의 급식

나이스(NEIS) 오픈 API를 활용하여 학교의 중식과 석식 정보를 실시간으로 보여주는 웹 페이지입니다.

##  주요 기능
- **실시간 급식 확인**: 매일의 중식 및 석식 메뉴와 칼로리를 확인합니다.
- **날짜 자동 업데이트**: 접속한 날짜를 자동으로 인식하여 표시합니다.
- **모바일 최적화**: 핸드폰에서도 보기 편한 다크 모드 디자인이 적용되어 있습니다.

##  시작하기
1. `config.js` 파일을 열어 본인의 **나이스 API 키**를 입력합니다.
2. `school.html` 파일을 브라우저로 실행하면 바로 확인할 수 있습니다.

##  보안 안내
- API 키가 포함된 `config.js` 파일은 `.gitignore`에 등록되어 있어, 깃허브에 업로드할 때 자동으로 제외됩니다. 개인 보안을 안전하게 유지할 수 있습니다.

##  사용 기술
- HTML / CSS (Vanilla)
- JavaScript (Fetch API)
- NEIS Open API
